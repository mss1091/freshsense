# FreshSense

**Stop Guessing. Start Sensing.**

FreshSense is an AI-powered IoT food freshness monitoring system that detects spoilage before it happens. This repository actively hosts the production marketing and product launch website.

## Project Structure
This is a high-performance, strictly zero-dependency Vanilla HTML5 and CSS3 static website.
- `/` - Root HTML pages optimized for automatic GitHub Pages deployment.
- `/assets/css/` - Global stylesheets.
- `/assets/images/` - Optimized graphic assets and photography.

## Deployment
This repository is configured for out-of-the-box deployment on **GitHub Pages**. Simply set GitHub Pages source to the `main` branch root folder. No `npm build` or external services required.
